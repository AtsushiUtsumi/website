使いかた(仮)

ひとまずsvelteで動作確認

                npm init svelte@next hoge

プロジェクトルートディレクトリ内でsugumiをクローンして起動

                (git clone https://github.com/AtsushiUtsumi/sugumi.git) && (cd sugumi) && (python main.py)

少しごにょごにょ
「.vscode」 ← いつも使っている設定ファイルが有ればプロジェクトルートにコピーしておく 「cli」ディレクトリ ← 全言語のアプリケーション層のメソッド呼び出しをCLI動作確認のため入れておく 「run.bat」 ← CLIでの動作確認のため上記「cli」ディレクトリ内のファイルを全て実行するバッチファイル
