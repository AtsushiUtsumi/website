使いかた(仮)

ひとまずsvelteで動作確認

                npm init svelte@next hoge

プロジェクトルートディレクトリ内でsugumiをクローンして起動

                (git clone https://github.com/AtsushiUtsumi/sugumi.git) && (cd sugumi) && (python main.py)

少しごにょごにょ
「.vscode」 ← いつも使っている設定ファイルが有ればプロジェクトルートにコピーしておく
「cli」ディレクトリ ← 全言語のアプリケーション層のメソッド呼び出しをCLI動作確認のため入れておく
「run.bat」 ← CLIでの動作確認のため上記「cli」ディレクトリ内のファイルを全て実行するバッチファイル

仕様(仮)

開発者にやってもらうことは以下の2つ
  1. 各レイヤーのシードファイルかディレクトリ的なものを用意する
  2. sugumiを起動「python sugumi main.py」
各レイヤーのシードファイルかディレクトリ的なものを用意する
「sugumi」内「main.py」を起動するとコード生成する
生成するのはDBはそのうちやろう...
